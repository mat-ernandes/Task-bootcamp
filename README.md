# Gerenciador De Tarefas - Task

O Task é um sistema web de gerenciamento de tarefas desenvolvido com o objetivo de auxiliar na organização, acompanhamento e controle de atividades em um ambiente colaborativo.

A aplicação permite o cadastro de tarefas com informações detalhadas, como título, descrição, responsável, prioridade, prazo e status, possibilitando uma visão clara do andamento das atividades. Além disso, o sistema oferece recursos de filtragem e visualização dinâmica, facilitando a gestão e tomada de decisões.

O projeto foi desenvolvido com foco na separação entre as camadas de apresentação e lógica, utilizando HTML, CSS e jQuery no front-end, com estrutura preparada para integração com PHP e banco de dados PostgreSQL no back-end.

## Tecnologias Front
- HTML
- CSS
- jQuery

## Funcionalidades
- Cadastro de tarefas
- Edição
- Exclusão
- Status
- Prioridade
- Responsável
- Prazo
- Filtros
- Cards de resumo
